게임 : 소환대전 어나더

기간 : 2010 ~ 2017

사용기술 : 블리자드 에디터, jass Script

프로젝트 url : 카페

소개 : 워크래프트3에서 플레이 가능한 AOS 장르의 유즈맵

  저는 어릴 적부터 게임을 좋아했습니다. 단순히 게임을 해보는 것뿐만 아니라 제가 원하는 게임을 만들기 위해 플래시, RPG game maker, 유니티와 같은 여러 개발도구와 그에 맞는 개발 언어, GUI 라이브러리, 플랫폼들을 사용해보았습니다.
 해당 작품은 워크래프트라는 게임의 유즈맵(사용자 정의 시나리오)입니다.   개발 과정중에 네이버 블로그와 카페를 통해서 사용자들과 의견을 나누고 이를 개발에 반영하였습니다.
  링크의 카페에서 게임의 스크린샷, 개발내용, 사용자들과 소통하며 즐겨온 기록등을 확인할 수 있습니다.

웹 : E-sports platform

기간 : 2019.04 ~ 2019.07

사용기술 : 백엔드 (Python-django), 프론트엔드(css,js,bootstrap)

프로젝트 url : 깃헙

소개 : 사용자가 팀을 만들고 대회를 생성/관리/참가 가능한 웹 프로젝트.

  사용자들은 원하는 게임의 대회를 만들거나 참가할수있습니다.
  저는 해당 프로젝트의 제품관리자 역활을 맡아 각 기능들에 대한 요구사항을 정의하고 구현에도 참여했습니다. 웹 프로젝트 개발과 관리자로써의 경험을 동시에 얻은 귀중한 시간이였습니다.
  -해당 프로젝트중 제가 직접 개발한 부분은 아래와 같습니다
  *이메일 인증을 통한 회원가입.
  *대회 신청시 자동으로 대진을 생성하고 이를 대진표로 시각적으로 보여주기.
  *팀을 만들거나 검색하여 참가하기.
  *연습경기를 만들거나 검색하여 참가하기.

안드로이드 : Move On

기간 : 2018.04 ~ 2018.07

사용기술 : Java, PHP

프로젝트 url : 깃헙

소개 : 지체장애인들을 위한 사용자 참여형 지도 어플리케이션.

  안드로이드 수업시간에 팀프로젝트로 진행한 지도 어플리케이션입니다.
사용자는 원하는 장소를 등록하여 해당 장소가 제공하는 서비스(장애인 화장실, 엘레베이터 유무 등)를 공유할 수 있으며, 등록된 장소까지의 길찾기 기능을 지원합니다.
  정보를 저장하기위해 사설서버를 기간임대하여 mysql과 php를 이용하였습니다.
